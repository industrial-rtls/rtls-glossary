RTLS Planner

Sewio's project design tool that streamlines RTLS deployments by optimizing anchor placement and system configuration before installation. Enables instant Dilution of Precision (DOP) simulation, supports 2D and 3D planning with CAD file import (DXF, PDF), calculates coverage and expected precision, and helps minimize project costs by preventing unplanned anchor deployment updates. Available exclusively to Sewio partners. Includes collaboration features for sharing designs with Sewio consultants.

RTLS Planner is Sewio Networks' planning and design tool for RTLS deployments, enabling system integrators and facility planners to design optimal anchor layouts, predict system performance, and plan implementations before physical installation. This pre-deployment tool addresses a critical challenge in RTLS projects: determining optimal infrastructure placement to achieve required coverage, accuracy, and reliability while minimizing costs. The software workflow begins with importing facility floor plans (CAD files, PDFs, or images) and defining the tracking area dimensions. Users then specify system requirements including desired positioning accuracy, coverage areas, expected tag density, and any special constraints (height restrictions, mounting limitations, power availability). The planning engine analyzes the facility layout considering obstacles, RF propagation characteristics, and anchor visibility requirements to recommend optimal anchor placement.

Key capabilities include: RF propagation modeling accounting for building materials, obstacles, and reflective surfaces, coverage analysis visualizing which areas have adequate anchor visibility for reliable positioning, accuracy prediction estimating expected positioning performance based on anchor geometry and environmental factors, and infrastructure optimization suggesting minimum anchor count while meeting requirements. Site survey integration allows importing actual RF measurements to refine propagation models and validate planned performance. RTLS Planner reduces deployment risks by identifying issues before installation: insufficient coverage areas, accuracy limitations from poor anchor geometry, capacity constraints, or unrealistic performance expectations. Export functions generate documentation for installation teams, project stakeholders, and system documentation. Using RTLS Planner in the pre-sales and project planning phases ensures deployments meet customer requirements, avoids costly redesigns during installation, and sets realistic performance expectations.
