RTLS Studio

Software & Data Management

Sewio's all-in-one software package for installation, deployment, maintenance, and control of the RTLS platform. Includes RTLS Manager (configuration), Sensmap (visualization), RTLS Monitor (performance monitoring), Open API documentation, and integration with myRTLS Insights and Sage Analytics. Provides unified interface for managing entire RTLS system remotely via web browser. Supports deployment via Docker, native Ubuntu, or VirtualBox. Latest version available through myRTLS Care subscription.

RTLS Studio is Sewio Networks' advanced development and customization environment for building tailored RTLS applications, dashboards, and integrations specific to customer requirements. While Sewio provides standard applications (Monitor, Manager, Planner), many industrial customers need specialized interfaces, custom workflows, or unique business logic that generic tools cannot address. RTLS Studio provides the framework, tools, and APIs for developing these custom solutions without requiring deep knowledge of positioning algorithms or low-level hardware interfaces. The development environment includes: comprehensive REST APIs for accessing real-time and historical position data, WebSocket interfaces for streaming position updates, software development kits (SDKs) for common programming languages (JavaScript, Python, .NET, Java), digital floor plan management APIs, geofence and zone management interfaces, and tag/asset configuration APIs. Pre-built UI components and libraries accelerate custom development: map viewers with position overlays, asset status widgets, alert management components, reporting templates, and data visualization libraries. The development environment supports rapid prototyping and iterative development, allowing proof-of-concept implementations to validate approaches before full-scale development. Customization examples include: modifying alert logic to implement company-specific escalation procedures, creating specialized reports matching existing business reporting formats, building integration bridges connecting RTLS with legacy systems lacking standard APIs, and developing industry-specific applications (automotive, aerospace, pharmaceutical) addressing unique requirements. RTLS Studio enables system integrators and industrial software developers to leverage Sewio's positioning infrastructure while creating differentiated solutions addressing specific customer needs. This extensibility makes Sewio's RTLS Platform suitable for diverse industrial applications beyond standard tracking scenarios.
