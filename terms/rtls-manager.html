RTLS Manager

RTLS Manager is Sewio Networks' core software platform for administering, configuring, and monitoring industrial RTLS deployments. This enterprise-grade management application serves as the central control point for the entire location system, providing tools for system administrators, integrators, and facility managers to optimize RTLS performance and manage operational aspects.

Key capabilities include: infrastructure management (configuring and monitoring anchors/readers, managing network connectivity, monitoring system health), tag administration (provisioning tags, assigning to assets or personnel, configuring transmission parameters, managing battery status), zone and geofence definition (creating virtual boundaries on digital floor plans, configuring alert rules and access permissions), user access control (role-based permissions for different organizational functions), positioning engine configuration (selecting algorithms, tuning parameters for accuracy vs. update rate tradeoffs), and system diagnostics (troubleshooting connectivity issues, analyzing signal quality, identifying interference sources). RTLS Manager integrates with Sewio's positioning engine and provides interfaces for site survey planning, anchor placement optimization based on facility layout, and RF environment analysis. Performance monitoring capabilities track key metrics including: positioning accuracy and availability, anchor uptime and connectivity, tag battery levels and estimated replacement dates, system capacity utilization (active tags vs. maximum supported), and network health. Alert management features notify administrators of system issues requiring attention: anchor offline conditions, communication failures, positioning engine errors, or capacity approaching limits. RTLS Manager is designed for 24/7 industrial operation with high availability architecture, automated backup capabilities, and disaster recovery features. The web-based interface enables remote administration and monitoring, critical for multi-site industrial enterprises.
